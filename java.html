Create Your Own Adventure Assignment

The Create Your Own Adventure assignment was our first Java assignment. It taught us how to print, and more importantly, it taught us if and else if statements. In addition, we learned the basics of different data types, such as integer and string. We also learned how to take keyboard inputs and use it in our adventure. The simple basics of java were taught in this assignment.

There are many different endings for this adventure.
